# Groupe de briand_c

Install criterion:

Mac OS X: 
brew install snaipe/soft/criterion

AUR: 
pacaur -S criterion

Ubuntu/Debian:
sudo add-apt-repository ppa:snaipewastaken/ppa

sudo apt-get update

sudo apt-get install criterion-dev

Compile:

`make re`

Compile for profiling:

`make profiling`

Run:

`./http-server`

Test:

- Compile:

`make test`

- Run:

`./http-server-test`
